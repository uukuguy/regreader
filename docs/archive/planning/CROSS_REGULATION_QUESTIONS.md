# 跨规程测试问题集

## 概述

本文档包含需要结合**安规2024**和**稳规2024**两个规程才能完整回答的测试问题。

### 规程说明

| 规程 ID | 全称 | 侧重点 |
|---------|------|--------|
| `angui_2024` | 2024年国调直调安全自动装置调度运行管理规定（第二版） | 装置配置、功能、操作管理 |
| `wengui_2024` | 2024年特高压互联电网稳定及无功电压调度运行规定（第二版） | 稳定限额、运行方式、控制要求 |

### 问题设计原则

- **稳规**提供：运行条件、稳定限额、控制要求
- **安规**提供：装置结构、动作逻辑、操作规程
- 两者结合才能给出完整答案
